# WebStudio Command Reference Guide

This reference guide provides a comprehensive list of commands available in WebStudio, categorized for easy navigation. Use these commands to enhance your web design and development projects.

## Design Commands

| Parameter/Command      | Type         | Description                                           | Example                      |
|------------------------|--------------|-------------------------------------------------------|------------------------------|
| `addElement`           | Function     | Adds a new design element to your project.           | `addElement('button')`      |
| `setBackgroundColor`   | Function     | Sets the background color of a selected element.     | `setBackgroundColor('#fff')`|
| `resizeElement`        | Function     | Resizes a selected element to specified dimensions.   | `resizeElement(100, 200)`   |
| `setFont`              | Function     | Applies a font style to a selected text element.     | `setFont('Arial', 16)`      |

## Layout Commands

| Parameter/Command      | Type         | Description                                           | Example                      |
|------------------------|--------------|-------------------------------------------------------|------------------------------|
| `createGrid`           | Function     | Creates a grid layout for your design.               | `createGrid(3, 4)`          |
| `alignElement`         | Function     | Aligns a selected element within its parent container.| `alignElement('center')`     |
| `setMargin`            | Function     | Sets the margin for a selected element.              | `setMargin(10)`             |
| `setPadding`           | Function     | Sets the padding for a selected element.             | `setPadding(15)`            |

## Interaction Commands

| Parameter/Command      | Type         | Description                                           | Example                      |
|------------------------|--------------|-------------------------------------------------------|------------------------------|
| `addEventListener`     | Function     | Attaches an event listener to a selected element.    | `addEventListener('click')`  |
| `removeEventListener`  | Function     | Removes an event listener from a selected element.   | `removeEventListener('click')`|
| `triggerEvent`         | Function     | Triggers a specified event on a selected element.    | `triggerEvent('submit')`    |
| `setAnimation`         | Function     | Applies an animation to a selected element.          | `setAnimation('fadeIn')`    |

## Customization Commands

| Parameter/Command      | Type         | Description                                           | Example                      |
|------------------------|--------------|-------------------------------------------------------|------------------------------|
| `customCSS`            | Function     | Applies custom CSS styles to your project.           | `customCSS('.myClass { color: red; }')` |
| `importComponent`      | Function     | Imports a reusable component into your project.      | `importComponent('navbar')` |
| `exportProject`        | Function     | Exports your project as a zip file.                  | `exportProject('myProject.zip')` |
| `setMetaTags`          | Function     | Sets meta tags for SEO optimization.                  | `setMetaTags({ title: 'My Site', description: 'Description here' })` |

Utilize these commands to maximize your efficiency and creativity in WebStudio. For further details, refer to the official documentation or community forums.
