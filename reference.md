# WebStudio Command Reference

This reference guide provides a comprehensive overview of commands and parameters in WebStudio. Use this to enhance your web design and development process.

## Design Tools

| Parameter/Command      | Type         | Description                                   | Example                        |
|------------------------|--------------|-----------------------------------------------|--------------------------------|
| `addElement`           | Function     | Adds a new design element to the canvas.    | `addElement('button')`        |
| `setBackgroundColor`   | Function     | Sets the background color of the selected element. | `setBackgroundColor('#FFFFFF')` |
| `resizeElement`        | Function     | Resizes the selected element to specified dimensions. | `resizeElement(200, 100)`     |
| `applyStyle`           | Function     | Applies a predefined style to the selected element. | `applyStyle('primary-button')` |

## Layout Management

| Parameter/Command      | Type         | Description                                   | Example                        |
|------------------------|--------------|-----------------------------------------------|--------------------------------|
| `createGrid`           | Function     | Creates a grid layout for the page.         | `createGrid(3, 2)`            |
| `addRow`               | Function     | Adds a new row to the current grid layout.  | `addRow()`                    |
| `setColumnWidth`       | Function     | Sets the width of a specific column in the grid. | `setColumnWidth(1, '50%')`    |
| `removeElement`        | Function     | Removes the selected element from the layout.| `removeElement('header')`     |

## Advanced Customization

| Parameter/Command      | Type         | Description                                   | Example                        |
|------------------------|--------------|-----------------------------------------------|--------------------------------|
| `customCSS`            | Function     | Applies custom CSS styles to the entire site. | `customCSS('.my-class { color: red; }')` |
| `addScript`            | Function     | Adds a JavaScript file to the project.      | `addScript('script.js')`      |
| `setSEO`               | Function     | Configures SEO settings for the website.     | `setSEO({ title: 'My Site', description: 'Best site ever' })` |
| `publishSite`          | Function     | Publishes the site to the web.               | `publishSite('mywebsite.com')` |

## User Management

| Parameter/Command      | Type         | Description                                   | Example                        |
|------------------------|--------------|-----------------------------------------------|--------------------------------|
| `addUser`              | Function     | Adds a new user to the project.              | `addUser('john@example.com')` |
| `removeUser`           | Function     | Removes a user from the project.             | `removeUser('john@example.com')` |
| `setUserRole`          | Function     | Assigns a role to a user in the project.    | `setUserRole('john@example.com', 'editor')` |
| `listUsers`            | Function     | Lists all users associated with the project. | `listUsers()`                 |

Utilize these commands to maximize your efficiency in WebStudio, outperforming competitors like Wix, Squarespace, and Webflow.
