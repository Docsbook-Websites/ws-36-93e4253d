# WebStudio Command Reference

This reference guide provides a comprehensive overview of commands and parameters available in WebStudio. Use this guide to enhance your website-building experience.

## General Commands

| Command              | Type    | Description                                      | Example                  |
|----------------------|---------|--------------------------------------------------|--------------------------|
| `createSite`         | Action  | Initiates the creation of a new website.        | `createSite("MySite")`  |
| `deleteSite`         | Action  | Permanently removes an existing site.            | `deleteSite("MySite")`   |
| `publishSite`        | Action  | Publishes your site to the web.                  | `publishSite("MySite")`  |
| `previewSite`        | Action  | Displays a live preview of your site.            | `previewSite("MySite")`  |

## Design Parameters

| Parameter            | Type    | Description                                      | Example                  |
|----------------------|---------|--------------------------------------------------|--------------------------|
| `setTheme`           | Action  | Applies a specific theme to your site.          | `setTheme("Modern")`     |
| `addLogo`            | Action  | Uploads a logo to your site header.             | `addLogo("logo.png")`    |
| `setFont`            | Action  | Changes the font style for your site text.      | `setFont("Arial")`        |
| `setColorPalette`    | Action  | Defines the color scheme for your site.          | `setColorPalette("blue")` |

## Content Management

| Command              | Type    | Description                                      | Example                  |
|----------------------|---------|--------------------------------------------------|--------------------------|
| `addPage`            | Action  | Creates a new page on your site.                | `addPage("About Us")`    |
| `editPage`           | Action  | Modifies content on an existing page.           | `editPage("Home")`        |
| `deletePage`         | Action  | Removes a page from your site.                  | `deletePage("Contact")`   |
| `addBlogPost`        | Action  | Adds a new blog post to your site.              | `addBlogPost("New Trends")` |

## SEO Settings

| Parameter            | Type    | Description                                      | Example                  |
|----------------------|---------|--------------------------------------------------|--------------------------|
| `setMetaTitle`       | Action  | Defines the title that appears in search results.| `setMetaTitle("My Business")` |
| `setMetaDescription` | Action  | Sets the description for search engines.        | `setMetaDescription("Best services in town.")` |
| `addKeyword`         | Action  | Adds keywords for SEO optimization.             | `addKeyword("web design")` |

## User Management

| Command              | Type    | Description                                      | Example                  |
|----------------------|---------|--------------------------------------------------|--------------------------|
| `addUser`            | Action  | Invites a new user to collaborate on your site. | `addUser("john@example.com")` |
| `removeUser`         | Action  | Revokes access for a user.                       | `removeUser("john@example.com")` |
| `setPermissions`      | Action  | Adjusts user permissions for site access.       | `setPermissions("editor")` |

Utilize this reference to navigate WebStudio's features effectively, ensuring a streamlined website creation process tailored to your needs.
